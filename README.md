# `asciidoctor-tei`

> Convert AsciiDoc documents into XML-TEI files,
> at least a subset used by [OpenEdition publishing](https://www.openedition.org/).

## Usage

### Command line

```bash
$ asciidoctor --require asciidoctor-tei -b tei ./docs/sample.adoc
```
